### Tasks

1. Create a database named **Employee**.
2. Create a table named **Tbl_Employee**.
3. Insert the following data into **Tbl_Employee**.

<div className="bg-background overflow-x-auto rounded-md">

| Employee ID | Name  | Position | Salary | Department |
| :---------: | :---- | :------- | :----- | :--------- |
| 101         | John  | Manager  | 25,000 | Sales      |
| 102         | Sarah | Analyst  | 30,000 | Finance    |
| 103         | Mike  | Developer| 35,000 | IT         |
| 104         | Vince | Assistant| 20,000 | HR         |

</div>

4. Update the Salary of the employee with **Employee ID 101** to **30,000**.
5. Add an Address column to the **Tbl_Employee** table.
6. Update the Address of the employees, setting it to "**Sumacab, Cabanatuan City**".
